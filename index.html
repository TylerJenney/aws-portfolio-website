How This Website Was Built

This website is built using AWS cloud services:

  • Amazon S3: Static website hosting
  • Route 53: Custom domain management
  • CloudFront: Global content delivery
  • AWS Lambda, API Gateway & DynamoDB: Serverless visitor counter
  • IAM (Identity and Access Management): Controls permissions for each service

This website is hosted on AWS using Amazon S3 for static site hosting, CloudFront for global content delivery and SSL security, and Route 53 for domain management. A serverless visitor counter is implemented using API Gateway, AWS Lambda (Python), and DynamoDB to track site visits. Security is enforced through IAM roles, bucket policies, and CORS settings. The resume download is served via CloudFront, ensuring fast and secure access. This architecture provides a scalable, cost-efficient, and high-performance web solution fully powered by AWS.

I will be constantly expanding this website with new features that use all sorts of AWS services in the future!
